# Vịt Quay Nhiều Vị - Studocu Downloader

Extension hỗ trợ xem và tải tài liệu miễn phí từ Studocu. Được phát triển bởi **AnhDT**.

## 🚀 Tính năng chính

- **Xem & Xóa Watermark**: Loại bỏ các lớp phủ và watermark chặn tài liệu, cho phép xem trọn vẹn nội dung.
- **Tải File PDF**: Tự động dàn trang và xuất tài liệu dưới dạng PDF chất lượng cao.
- **Lưu thành ảnh (PNG)**: Chụp và lưu riêng từng trang tài liệu đang hiển thị.

## ⚙️ Hướng dẫn cài đặt (Chrome / Edge)

Do extension tải dạng mã nguồn, vui lòng cài đặt qua chế độ Developer mode.

1. Tải toàn bộ mã nguồn về máy (hoặc git clone).
2. Mở trình duyệt và truy cập trang quản lý extension:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. Bật **Chế độ dành cho nhà phát triển (Developer mode)** ở góc trên cùng.
4. Chọn nút **Load unpacked** (Tải tiện ích đã giải nén).
5. Trỏ tới thư mục `tool_studocu` trong mã nguồn và chọn **Select Folder**.
6. Ghim extension lên thanh công cụ (toolbar) để dễ dàng thao tác.

## 📖 Hướng dẫn sử dụng

1. Truy cập vào tài liệu cần xem trên [Studocu.com](https://www.studocu.com/) hoặc [Studocu.vn](https://www.studocu.vn/).
2. **Quan trọng:** Cuộn chuột từ từ xuống cuối tài liệu để đảm bảo toàn bộ các trang đã được load.
3. Bấm vào icon của extension trên thanh công cụ.
4. Lựa chọn chức năng:
   - **Tải File PDF**: Xuất toàn bộ bài đã load thành file PDF.
   - **Xem file & Xóa Watermark**: Xóa khung chặn premium, loại bỏ mờ trang.
   - **Lưu thành ảnh**: Tải trang tài liệu đang hiển thị thành file `.PNG`.
